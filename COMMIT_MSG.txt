Bilancio CEE: autonomous calculation engine (writes BILNEW)

The web now recalculates the CEE balance itself (no longer needs the gestionale).
Safe because BILNEW is volatile: fully rewritten on each recalculation.

Faithful port of ceecont.PRG (per resources/cee/README.md):
- Phase 1+2: for each U_INT_TT mapping, account balance = CON_IMP_D - CON_IMP_A
  (+forecast if requested); target row = INT_CODRIG if balance >= 0 else INT_CODRIA
  when present ("banche in passivo"); accumulate into the row's CORRENTE. Missing
  accounts / missing target rows are warned and skipped.
- Phase 2b: ABS on economic rows (VAL(codrig) >= 21600).
- Phase 3: totals from U_COR_TT signed edges (COR_RIGA order); a missing
  confluence row aborts with nothing written ("aggiornamento annullato").
- In-memory compute, single batch write at the end; robust numeric code keys.
