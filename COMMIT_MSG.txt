Web activity log + bilancio elaboration time; drop redundant section titles

(1) Removed the redundant section-title header from the balance section
    fragments: the sub-tabs (Attivita/Passivita, Costi/Ricavi) already title them.
(2) The controller measures the balance computation time (elaborazioneMs) and the
    page shows it after Elabora ("Elaborato in X,XX s (N ms)").
(3) Web-only system activity log: ActivityLoggingInterceptor records per request
    user, method, uri+query, HTTP status and duration (ms); registered in
    WebMvcConfig for /** excluding static assets. logback-spring.xml gets a
    dedicated ACTIVITY_FILE appender (aquarius-web-activity.log, daily+10MB
    rotation, 90 days, gzip) and an AQUARIUS_ACTIVITY logger (additivity=false).
    File-based, no DB overhead; an in-app viewer + aq_web_activity_log table can
    follow if browsing from the app is needed.

Version 0.14.0. Sandbox checks only.
