Move identity block to top and add collapsible sidebar

Two sidebar UI changes, confined to layout.html.

1. The tenant + user + fiscal-year block moves from the footer to a new
   .sidebar-identity section between the brand and the menu. The footer now
   keeps only the Esci button and the app version.

2. The sidebar can collapse to an icons-only rail. A toggle button in the
   brand row switches a .collapsed class: width 260px -> 64px with the main
   content margin following (animated). Collapsed mode hides all text labels,
   centers icons, hides nested submenus, and shows tooltips (th:title) on each
   item. The state is saved in sessionStorage and restored on load.

No server-side or behavioural change.

Verification: node --check on the layout script, aside/main confirmed as
direct siblings, git apply --check clean on a fresh clone of HEAD cadec67.
Visual behaviour to confirm on deploy.
