Add supplier registry (anagrafica fornitori)

Replica of the VFP form MENU_FOR000, reusing the customer framework
(form-shell, FormTab, whitelist copy).

Fields were taken from MENU_FOR000.SCT and cross-checked against the schema
(U_FOR_AN, 195 columns): 44 verified core fields across 7 active tabs
(Anagrafica, Indirizzo e contatti, IVA, Commerciali, Dati contabili, Dati
bancari, Rif. fornitore). No invented columns.

- Supplier entity (U_FOR_AN, @DynamicUpdate); key id_unique / FOR_CODSOC /
  FOR_CODCON immutable from web.
- SupplierRepository with paged search on name/code/VAT.
- SupplierController: /fornitori list, detail, edit, save with a
  whitelist copyEditableFields.
- fornitori/{list,detail,edit}.html: list and full-page detail cards
  (rendered only when populated), edit on the shared form-shell fragments.
- MenuService maps menu_for000 -> /fornitori.

and supplier.X maps to an entity property, div balance list 5/5 detail 32/32
