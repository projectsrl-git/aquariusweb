Document row detail: expandable rows + comment CLOB (invoices)

Cesello on the document detail view, done on invoices first as the reference
(to validate the untestable part — reading the ORD_NOTE text/CLOB via jTDS —
before replicating to orders/DDT/proforma).

- Comment rows (*** COMMENTO *** on ORD_DESART) now show the real comment body
  from the ORD_NOTE CLOB, matching FATTURAZIONE_ELETTRONICA_LIB.
- Rows are clickable to expand a grouped detail panel (UM/weight, dimensions,
  source order, destination, notes) shown only when populated.
- InvoiceRow maps the curated extra fields + ORD_NOTE/ORD_NOTEBO
  (columnDefinition="text", read-only). New shared fragment fragments/doc-row.html
  (detailRow + toggleScript), reusable by the other document types.
- Version 0.3.1 (patch: refinement of existing modules).

Sandbox checks only (no Maven). Confirm on deploy that the text columns read as
String via jTDS/Hibernate on SQL Server 2012; if OK, replicate to orders/DDT/
proforma. If Hibernate objects to the text type, @Lob is the fallback.
