Bilancio phase 2: month range + inline search + progressive loader timer

- Month range (mese dal/al): options form gains "Mese dal"/"Mese al" (1..12,
  default 1/12). New findBilancioPeriodo() aggregates the same movements filtered
  by month (SUBSTRING(registrationDate,6,2) BETWEEN dal AND al). Full year (1..12)
  keeps the original query so the default result is unchanged; a "Periodo mese X-Y"
  badge shows when narrowed. NOTE: this is the movement period only - opening
  balances for dal>1, previous-year/forecast options still to come and to be
  reconciled with the gestionale.
- Inline search: a "search account or description" box highlights matching rows,
  counts them, activates the tab of the first match and scrolls to it.
- Progressive timer in the global loading overlay: shows elapsed seconds ticking
  up while waiting.

Version 0.15.0. Sandbox checks only.
